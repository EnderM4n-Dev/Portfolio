# EnderDev — Portfolio

پورتفولیوی شخصی محمدحسین نوری (EnderDev) — یک سایت چندصفحه‌ای دوزبانه (فارسی/انگلیسی) با تم شب/روز و چهار پالت رنگی.

## صفحات
- `index.html` — ریدایرکت به صفحه‌ی خانه
- `Home.html` — صفحه‌ی اصلی
- `Projects.html` — پروژه‌ها
- `About.html` — درباره من
- `Contact.html` — تماس

## فایل‌ها
- `styles.css` — استایل کل سایت
- `palette.js` — سیستم پالت رنگی + تم شب/روز (روی همه‌ی صفحات)
- `app.js` — تعامل‌های مشترک (منو، سوییچ زبان)
- `home-fx.js` — افکت‌های صفحه‌ی خانه
- `projects-bg.js` — انیمیشن زمینه (WebGL)
- `gallery.js` — گالری/لایت‌باکس صفحه‌ی پروژه‌ها
- `assets/` — تصاویر و لوگوها

## اجرا
چون سایت کاملاً استاتیکه، کافیه فایل‌ها رو روی هر هاست استاتیک (مثل GitHub Pages) بذاری. فایل `.nojekyll` برای GitHub Pages موجوده.

برای اجرای محلی، یک سرور ساده کافیه:

```bash
python3 -m http.server
```

سپس `http://localhost:8000` را باز کن.
